# OPEN-BLACKFIRE

Vendor-agnostic Quantum Computing framework for Common Lisp.

## Support

This library is built and tested on macOS with the following Lisp implementations:

- LispWorks 8.0.0 (64-bit)
- SBCL 2.2.4
- Clozure-CL 1.12.1
- GNU CLISP 2.49.92
- ECL 21.2.1
- ABCL 1.9.0
